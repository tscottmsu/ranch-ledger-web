# Sprint 3: Guests, Reservations, and Brand Theme

## Goal

Add guest/reservation management and apply the Ranch Ledger visual identity.

## Included

- Guests CRUD
- Reservations CRUD
- Link guests to reservations
- Dashboard updates for guest/reservation counts
- Ranch Ledger theme inspired by the approved logo direction
- Update app branding, colors, logo treatment, favicon/app icon placeholders

## Branding Direction

Use the uploaded Ranch Ledger logo concept as inspiration.

Visual style:
- Modern ranch operations platform
- Burnt orange accent
- Dark charcoal text
- Warm off-white backgrounds
- Clean badge/shield-inspired identity
- Professional outdoor SaaS feel
- Not cartoon western

## Not Included

- Activity scheduling
- Assignment engine
- Maps
- Live GPS
- Emergency beacon

## Acceptance Criteria

- Ranch Administrator can add, edit, list, and archive reservations.
- Ranch Administrator can add, edit, list, and archive guests.
- Guests may optionally belong to a reservation.
- Guest status is tracked.
- Dashboard includes guest and reservation setup/progress data.
- Left sidebar and dashboard use the Ranch Ledger brand theme.
- TypeScript, ESLint, and production build pass.
